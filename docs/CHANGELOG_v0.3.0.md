# CutClip v0.3.0 — Professional Edition

## Identidad visual
- Nuevo icono propio para la ventana, barra de tareas, bandeja, EXE e instalador.
- Créditos visibles: **Diseñado y desarrollado por Bryant Brugal**.
- Ventana **Acerca de** con versión, build, descripción, copyright y copia de diagnóstico.
- Metadatos profesionales del ejecutable.

## Bandeja del sistema
- Clic o doble clic en el icono restaura la ventana.
- Estado dinámico: verde conectado, amarillo reconectando y rojo desconectado.
- Acciones rápidas para guardar clips, abrir carpetas, configurar, reconectar, ver información y salir.
- Cerrar con la X minimiza a la bandeja de forma segura.

## Notificaciones
- Avisos nativos al guardar clips.
- Avisos de desconexión y reconexión de OBS.
- Opción para desactivar las notificaciones.

## Inicio y segundo plano
- Inicio opcional con Windows.
- Modo silencioso para arrancar directamente en la bandeja.
- Argumento `--silent` compatible con accesos directos e instalador.

## Interfaz
- Tarjetas principales con apariencia más cercana a botones reales.
- Botón de contraseña sustituido por icono de ojo.
- Configuración plegable con flechas.
- Barra de resumen del último clip, hora y duración.
- Botón Acerca de en la pantalla principal.

## Distribución
- Especificación de PyInstaller incluida.
- Script de compilación del EXE actualizado.
- Proyecto de Inno Setup y script para compilar el instalador.
